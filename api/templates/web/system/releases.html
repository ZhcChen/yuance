发布约束

  • 新建版本默认是草稿，上传至少一个安装包后才能发布。
  • 一个版本可以包含多个平台产物：Windows、macOS、Linux、Android、iOS。
  • 安装包通过服务端临时签名地址上传，不在页面暴露长期 OSS 凭证。
